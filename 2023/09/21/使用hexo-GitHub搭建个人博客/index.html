4.搭建GitHub Page仓库并完成配置

首先需要在Github上注册账号

在点击账户旁边的 + ，再点击New repository新建仓库，按自己的喜好配置完后点击Create repository完成仓库的创建。

进入到新建的仓库页面后，点击导航栏里的Settings，侧边栏选择Page。记住自己的仓库的site网址。修改分支Branch从main改为master。

5.本地配置GitHub账户实现免密登陆

打开Git命令窗口，根据自己注册的GitHub账户设置用户名和邮箱。

创建SSH密钥

输入 ssh-keygen -t rsa -C "GitHub 邮箱"，然后一直回车。

到用户目录（一般在C:<用户名>下），记事本打开公钥 id_rsa.pub 文件并复制里面的内容。

登陆 GitHub ，进入 Settings 页面，选择左边栏的 SSH and GPG keys，点击 New SSH key。

Title 随便取个名字，粘贴复制的 id_rsa.pub 内容到 Key 中，点击 Add SSH key 完成添加。

打开 Git Bash，输入 ssh -T git@github.com 出现 “Are you sure you want to connecting"，输入 yes 回车确认。

显示"Hi! You' ve successfully authenticated",则连接成功。
